Lista 2
1. Crie um programa em Java que declare um array de inteiros e imprima seus elementos.
2. Crie um programa em Java que preencha um array com os quadrados dos números de 1 a N, onde N é um valor informado pelo usuário.
3. Crie um programa em Java que preencha um array com os números em ordem inversa.
4. Crie um programa em Java que encontre o maior e o menor número em um array e imprima-os na tela.
5. Crie um programa em Java que encontre o segundo maior elemento em um array.
6. Crie um programa em Java que calcule e mostre a soma de todos os elementos em um array.
7. Crie um programa em Java que calcule a soma dos elementos em posições pares de um array.
8. Crie um programa em Java que multiplique cada elemento de um array por um valor específico. Mostre o antes e o depois.
9. Crie um programa em Java que conte e imprima o número de elementos pares em um array.
10. Crie um programa em Java que substitua todos os elementos negativos em um array por zero.
11. Crie um programa em Java que encontre e escreva a média dos elementos de um array.
12. Crie um programa em Java que verifique e informe se um determinado valor está presente em um array.
13. Crie um programa em Java que copie os elementos de um array para outro. Imprima-os para mostrar que estão iguais.
14. Crie um programa em Java que ordene os elementos de um array em ordem crescente.
15. Crie um programa em Java que preencha um array com números aleatórios e, em seguida, ordene esses números de forma decrescente.
16. Crie um programa em Java que encontre e imprima o índice da primeira ocorrência de um elemento específico em um array. Caso ele não esteja deve-se imprimir -1.
17. Crie um programa em Java que verifique se dois arrays são iguais.
18. Crie um programa em Java que multiplique os elementos de dois arrays e armazene o resultado em um terceiro array.
19. Crie um programa em Java que inicialize um array de strings e depois inverta a ordem dos elementos do array. Imprima o antes e o depois.
20. Crie um programa em Java que remova todas as ocorrências de um texto específico em um array de strings.
21. Crie um programa em Java que substitua todas as vogais em um array de caracteres por asteriscos.
22. Crie um programa em Java que conte e imprima o número de vogais em um array de caracteres.
23. Crie um programa em Java que crie um novo array a partir de outros dois arrays.
24. Crie um programa em Java que insira um novo elemento em uma posição específica de um array. O usuário deve informar a posição e o elemento.
25. Crie um programa em Java que verifique se um array é um palíndromo.
26. Crie um programa em Java que gere um array com os N primeiros números primos.
27. Crie um programa em Java que preencha um array com os números perfeitos menores que N.
28. Crie um programa em Java que preencha um array com os primeiros N números da sequência de Fibonacci.
29. Crie um programa em Java que preencha um array com os N primeiros termos da sequência de Lucas.
30. Crie um programa em Java que remova os elementos repetidos de um array.
31. Crie um programa em Java que rotacione os elementos de um array para a esquerda.
32. Crie um programa em Java que implemente uma busca binária em um array ordenado.
33. Crie um programa em Java que adicione um novo elemento ao final de um array. Imprima o antes e o depois.
34. Crie um programa em Java que insira um elemento em uma posição específica de um array. Aumentando o tamanho do array. 
35. Crie um programa em Java que remova um elemento de uma posição específica de um array. Diminuindo o tamanho do array.
